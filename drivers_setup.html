Drivers Management Setup

Follow these steps to correctly set up the Drivers Management page

Quick Setup

Run the setup script to automatically create and populate the drivers database:

Run Setup Script

After running the setup, you'll be redirected to the Drivers page.

Troubleshooting

If you're experiencing issues, run the debug tool to check your database connection:

Run Debug Tool

The debug tool will check your database connection and show detailed information.

Manual Setup

If you prefer to set up manually, follow these steps:

  1. Make sure MySQL server is running
  2. Check that the database credentials in api/config.php are correct
  3. Create a database named traffic_management
  4. Run the SQL queries in database.sql to create tables
  5. Ensure PHP has permissions to access the database
